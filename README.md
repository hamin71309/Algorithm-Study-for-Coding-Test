# Algorithm Study for Coding Test
 
## 알맹이 스터디

"이것이 취업을 위한 코딩 테스트다 with 파이썬" 을 가지고 시작한 알고리즘 스터디 입니다.  
[책](http://www.yes24.com/Product/Goods/91433923)/[코드](https://github.com/ndb796/python-for-coding-test)

### 스터디 참여하는 방법

1. 알고리즘을 이해한다.
2. 책에 제공된 Python 코드를 이해한다.
3. C언어로 코드를 짜본다.
4. 파일명은 `문제번호_작성자.c`로 한다.
5. Commit message는 `문제이름`으로 통일한다. (ex: `거스름돈 문제`)
6. PR(Pull Request)명은 `작성자`로 통일한다.한다.
